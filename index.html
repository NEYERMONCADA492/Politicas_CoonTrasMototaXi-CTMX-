Política de Privacidad - Coontras Mototaxi

Esta aplicación recopila datos como ubicación, nombre y contacto con el fin de prestar el servicio de transporte.

La ubicación se usa únicamente para conectar usuarios con conductores cercanos.

No compartimos información con terceros.

Los datos pueden ser eliminados desde la aplicación en cualquier momento.

Para solicitudes de eliminación de datos, contáctanos.
